# Project Overview

This project represents the front end of a website that is a hypothetical design for a French café named LeS Vince Café.

The website is developed using only HTML and CSS and is intended for educational and demonstration purposes.

#Technologies Used

HTML

CSS (Internal CSS only)

No external CSS files or JavaScript are used in this project.


#Project Structure

Total Pages: 10 HTML web pages

Styling: Internal CSS embedded within each HTML file

Assets: Image files used throughout the website


#How to Run the Project

To execute the project correctly:

Download or clone the repository.

Ensure all HTML files and image files are kept in the same folder structure.

Open any HTML file in a web browser (recommended: home.html).


#Missing pages or images may cause broken links or display issues.
